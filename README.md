# Student Result Management System

A professional Full Stack Student Result Management System developed using:

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- MySQL

---

# Features

## Authentication System

- Admin Login
- Student Login
- Session Management
- Logout System
- Forgot Password Page

---

## Student Management

- Add Students
- View Students
- Manage Student Records

---

## Subject Management

- Add Subjects
- Assign Subjects to Classes
- Semester Management

---

## Result Management

- Automatic Result Calculation
- Percentage Calculation
- Grade Calculation
- Pass / Fail Status
- SGPA & CGPA Calculation

---

## Report Card

- Professional Report Card
- Download Report Card
- Print Report Card

---
